# Bowling League Tracker

Mountain Lakes Men's Bowling League season tracker.

Replaces/augments the Excel scoring workbook with a proper application for score entry, handicap calculation, standings, and prize tracking.

## Season Info

- League: Mountain Lakes Men's Bowling League
- Season: 2025-2026 (23 weeks, October – March)
- Teams: 4 (Lewis, Ferrante, Belyea, Mancini)
- Bowlers: ~65 (active + inactive)

## Handicap Formula

```
handicap = INT(0.9 * (200 - current_average))
```
